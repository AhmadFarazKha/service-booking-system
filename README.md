# AI Service Manager 2026 - Multilingual Trade Platform 🛠️

A modern, high-performance Service Management System built with **Laravel 11**, specifically engineered to bridge the gap between skilled tradespeople (Electricians, Plumbers, AC Technicians) and residents in the **Gulf/MENA region**.



## 🌟 Key Features
* **Multilingual Architecture**: Seamlessly localized in **English, Arabic, and Urdu** to serve a diverse population of locals and expats in the Gulf.
* **Service Lifecycle Management**: A robust CRUD system to add, edit, and manage trade services with precision.
* **Rich Data Structure**: Supports complex service attributes including **Title, Detailed Description, Price, and Skill-tagging**.
* **Smart Filtering & Tags**: Uses dynamic comma-separated parsing to display professional skill badges for each service.
* **Modern UI/UX**: Built with **Tailwind CSS**, featuring a responsive, clean, and intuitive dashboard layout.
* **Scalable Tech Stack**: Powered by Laravel 11, ensuring security and high performance under load.

## 🛠️ Technical Stack
* **Framework**: Laravel 11
* **Frontend**: Blade Templating Engine + Tailwind CSS
* **Database**: MySQL (Optimized with dynamic skill-tag support)
* **Storage**: Local/Cloud Storage for professional service imagery

## 🚀 Getting Started
1. **Clone the repository**:
   ```bash
   git clone [https://github.com/AhmadFarazKha/service-booking-system.git](https://github.com/AhmadFarazKha/service-booking-system.git)
```

2. **Install dependencies** :
   **Bash**

```
   composer install
   npm install
```

2. **Configure Environment** :
   Copy `.env.example` to `.env` and configure your database settings.
3. **Run Migrations** :
   **Bash**

```
   php artisan migrate
```

2. **Storage Link** :
   **Bash**

```
   php artisan storage:link
```

2. **Launch** :
   **Bash**

```
   php artisan serve
```
